# Global Higher Education Analysis

## Overview

This project analyzes global higher education performance using Microsoft Excel, Pivot Tables, Pivot Charts, and interactive dashboarding techniques. The dashboard provides insights into university rankings, educational quality, citation performance, alumni outcomes, and regional trends across institutions worldwide.

The objective of the project is to identify high-performing regions and universities while exploring relationships between key academic performance indicators.

---

## Dataset Summary

The dataset contains information on universities from around the world, including:

* University rankings
* Countries and regions
* Quality of education scores
* Citation scores
* Influence scores
* Alumni performance metrics
* Rank tiers

### Key Statistics

* **1,000 Institutions**
* **61 Countries**
* **6 Global Regions**
* **4 University Rank Tiers**

---

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Institutions
* Total Countries
* Highest Ranked Region
* Highest Ranked University

### Interactive Filters

* Region Slicer
* Rank Tier Slicer

### Visualizations

#### Average Score by Region

Compares overall performance across global regions.

#### Top 10 Universities by Quality of Education Score

Highlights the highest-performing universities based on educational quality.

#### Average Citation Score vs Average Influence Score

Explores the relationship between research influence and citation performance.

#### Alumni Score vs Quality of Education Score

Examines how educational quality relates to alumni outcomes.

---

## Key Insights

* North America emerged as the highest-performing region in the dataset.
* Harvard University ranked as the top institution based on the selected metrics.
* Higher influence scores generally corresponded with stronger citation performance.
* Educational quality showed a positive relationship with alumni success.
* Significant performance differences exist across regions and ranking tiers.
